# 🍠 GGM 개발 폴더 안내

## 이 폴더는?
개발 공부용 프로젝트 폴더예요. 실제 서비스처럼 만들어보면서 Next.js와 Supabase를 배우는 공간이에요.

## 기술 스택
- **Next.js** — 웹사이트를 만드는 프레임워크 (React 기반)
- **Supabase** — 로그인, 데이터베이스, 파일 저장 등을 담당하는 백엔드 서비스
- **Vercel** — 웹사이트를 인터넷에 올리는(배포) 서비스

## Claude에게 부탁하는 행동 방식

### 설명 방식
- 개발 용어는 **비개발자도 이해할 수 있게** 쉽게 풀어서 설명해줘
  - 예: "컴포넌트" → "화면 조각", "라우팅" → "페이지 이동", "쿼리" → "데이터 검색"
- 지금 **무엇을 하고 있는지** 자주 중간중간 알려줘
  - 예: "이제 로그인 화면을 만들고 있어요", "데이터베이스에 테이블을 추가하는 중이에요"

### 작업 방식
- 사용자가 **지시한 것만** 단계적으로 수행해 (추가 기능 임의로 넣지 않기)
- 할 수 있는 건 최대한 직접 작업해서 완성해줘
- 꼭 사람이 직접 해야 하는 것(클릭, 설정, 결제 등)은 **알기 쉽게 단계별로 안내**해줘
  - 예: "1. 이 링크를 열어주세요 → 2. ○○ 버튼을 눌러주세요 → 3. ○○을 입력하세요"

### 작업 완료 후
- 개발이 끝나면 **항상 브라우저 미리보기를 열어서** 바로 확인할 수 있게 해줘
- 잘 동작하는지 직접 눌러보며 테스트하고 결과를 알려줘

### Next.js 버전 주의
- 이 프로젝트는 **Next.js 16** 을 사용해요 (기존 버전과 다른 점이 있어요)
- `middleware.ts` 대신 `proxy.ts` 를 사용해야 해요
- 코드를 작성하기 전에 `node_modules/next/dist/docs/` 안의 문서를 먼저 확인해줘
